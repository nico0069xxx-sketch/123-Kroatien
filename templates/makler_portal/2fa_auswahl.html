Sicherheit einrichten

Warum Zwei-Faktor-Authentifizierung?

Die 2FA schuetzt Ihr Konto zusaetzlich zum Passwort. Selbst wenn jemand Ihr Passwort kennt, kann er sich ohne den zweiten Faktor nicht einloggen.

Waehlen Sie Ihre bevorzugte Methode:

Authenticator App

Nutzen Sie Google Authenticator, Microsoft Authenticator oder eine aehnliche App.

So geht's:
1. App herunterladen (kostenlos)
2. QR-Code mit der App scannen
3. 6-stelligen Code eingeben
Empfohlen - sehr sicher!
QR-Code scannen
E-Mail Code

Sie erhalten bei jedem Login einen 6-stelligen Code per E-Mail.

So geht's:
1. Login mit Benutzername & Passwort
2. Code wird an Ihre E-Mail gesendet
3. Code eingeben (5 Min. gueltig)
Einfach - keine App noetig
E-Mail nutzen
